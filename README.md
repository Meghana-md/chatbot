# RAG QnA Chatbot

This is a simple AI chatbot built using Streamlit.

It allows you to:
- Add (ingest) a document or text
- Store it in a vector database (ChromaDB)
- Ask questions about that document
- Get answers based on the document content

The chatbot does NOT guess — it answers using the stored document.

✅ How to Use

Open the app in your browser

Go to the Ingest page → add your document

Go to the Chatbot page → ask questions

The bot answers using your document
